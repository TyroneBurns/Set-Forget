Set & Forget
A live paper-trading dashboard using a Hidden Markov Model to estimate whether the market is bullish, bearish, or choppy before making or skipping trades.

Set & Forget uses a Hidden Markov Model to estimate whether the market is currently in a bull, bear, or chop regime.

It looks at recent momentum and volatility, updates the probability of each regime on every run, then compares signal strength against a required threshold.
