# Set & Forget v3

This build includes:

- candle-based HMM regime engine
- in-app explainer for HMM and future machine learning
- signal quality score
- adaptive confidence thresholds
- watchlist ranked by expected edge
- 1 day and 7 day percentage returns
- manual and auto paper trading
- Ably token route
- mobile-first dark layout

## Env vars

```env
ABLY_API_KEY=your_ably_api_key_here
APP_NAME=Set & Forget
DEFAULT_PAIRS=BTCUSDT,ETHUSDT,SOLUSDT,BNBUSDT,XRPUSDT
DEFAULT_TIMEFRAME=15m
```
